# Expense Solution
End-to-End analytic solution for expense monitoring.

## Purpose
- Monitor Financial Performance
- Enchance Budget Management
- Forecast Spending
- Provide Stakeholders with insights to make operational or strategic decision

## Existing Issue
- Manual data massage was done in excel to create a finalise breakdown of expenses.
- Annual budget is break down and calculated using excel.
